## 水面保洁详情页接口：

### 接口地址：/new/water_surface/detail/

### 请求方式：get

### 请求参数：

| 参数名 | 类型   | 是否必传 | 命名     | 备注 |
| ------ | ------ | -------- | -------- | ---- |
| userId | INT    | 是       | 用户ID   |      |
| cityId | INT    | 是       | 城市ID   |      |
| pId    | STRING | 否       | 省份ID   | 筛选 |
| aiType | STRING | 否       | AI类别   | 筛选 |
| sTime  | STRING | 否       | 开始时间 | 筛选 |
| eTime  | STRING | 否       | 结束时间 | 筛选 |
