## 创建水面保洁：

### 接口地址：v1/admin/clean/

### 请求方式：POST

### 请求参数：

| 参数名       | 类型   | 是否必传 | 命名         | 备注 |
| ------------ | ------ | -------- | ------------ | ---- |
| category     | INT    | 是       | 保洁分类     |      |
| cleaningType | INT    | 是       | 保洁类型     |      |
| areas        | STRING | 否       | 保洁面积     |      |
| distance     | STRING | 否       | 保洁距离     |      |
| weight       | STRING | 否       | 垃圾重量(kg) |      |
| sTime        | STRING | 否       | 保洁开始时间 |      |
| eTime        | STRING | 否       | 保洁结束时间 |      |
| provinceId   | INT    | 否       | 省份         |      |
| cityId       | INT    | 否       | 城市         |      |
| areaId       | INT    | 否       | 区           |      |
| townId       | INT    | 否       | 城镇         |      |
| villageId    | INT    | 否       | 乡           |      |
| shipId       | INT    | 是       | 船ID         |      |
| watersId     | INT    | 是       | 水域ID       |      |

### 请求示例:

```bash
{
    "category":11,
    "cleaningType":1,
    "areas": "666",
    "distance": "123",
    "weight": "1232",
    "sTime": "2021-08-01",
    "eTime": "2021-09-02",
    "provinceId": 1,
    "cityId":1,
    "areaId":1,
    "watersId":1,
    "shipId":2
}
```

### 响应

#### 200

```bash
{
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```
