## 创建预警：

### 接口地址：v1/admin/alarm/

### 请求方式：POST

### 请求参数：

| 参数名      | 类型   | 是否必传 | 命名         | 备注          |
| ----------- | ------ | -------- | ------------ | ------------- |
| eventType   | INT    | 是       | 事件类型     |               |
| address     | STRING | 否       | 事件地址     |               |
| lng         | STRING | 否       | 经度         |               |
| lat         | STRING | 否       | 纬度         |               |
| sTime       | STRING | 否       | 保洁开始时间 |               |
| eTime       | STRING | 否       | 保洁结束时间 |               |
| provinceId  | INT    | 否       | 省份         |               |
| cityId      | INT    | 否       | 城市         |               |
| areaId      | INT    | 否       | 区           |               |
| townId      | INT    | 否       | 城镇         |               |
| villageId   | INT    | 否       | 乡           |               |
| shipId      | INT    | 是       | 船ID         |               |
| watersId    | INT    | 是       | 水域ID       |               |
| isReported  | INT    | 是       | 是否已报     | 0:未报 1:已报 |
| ownerUserId | INT    | 是       | 业主ID       |               |

### 请求示例:

```bash
{
    "eventType":1,
    "address": "xxxx",
    "lng": "1231",
    "lat": "123",
    "sTime": "2021-08-01",
    "eTime": "2021-08-02",
    "provinceId": 1,
    "cityId":1,
    "areaId":1,
    "watersId":2,
    "shipId":2,
    "ownerUserId": 1,
    "isReported": 1
}
```

### 响应

#### 200

```bash
{
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```

## 创建造船信息：

### 接口地址：v1/admin/build/ship/

### 请求方式：POST

### 请求参数：

| 参数名     | 类型   | 是否必传 | 命名         | 备注          |
| ---------- | ------ | -------- | ------------ | ------------- |
| deviceNum  | STRING | 是       | 设备编号     |               |

| address    | STRING | 否       | 地址         |               |
| isReported | INT    | 否       | 是否已报     | 0:未报 1:已报 |
| sTime      | STRING | 否       | 保洁开始时间 |               |
| eTime      | STRING | 否       | 保洁结束时间 |               |
| userId     | INT    | 是       | 运维人员     |               |

```bash

    "safeType": "123132",
    "address": "科技",
    "isReported": 0,
    "sTime": "2021-08-13",
    "eTime": "2021-08-15",
```
