## 创建预警：

### 接口地址：v1/admin/alarm/

### 请求方式：POST

### 请求参数：

| 参数名       | 类型   | 是否必传 | 命名         | 备注          |
| ------------ | ------ | -------- | ------------ | ------------- |
| aiPId        | INT    | 是       | 一级事件类型 |               |
| aiSId        | INT    | 是       | 二级事件类型 |               |
| aiTId        | INT    | 是       | 三级事件类型 |               |
| address      | STRING | 否       | 事件地址     |               |
| lng          | STRING | 否       | 经度         |               |
| lat          | STRING | 否       | 纬度         |               |
| sTime        | STRING | 否       | 保洁开始时间 |               |
| eTime        | STRING | 否       | 保洁结束时间 |               |
| isReported   | INT    | 是       | 是否已报     | 0:未报 1:已报 |
| videoAddress | STRING | 否       | 视频地址     |               |
| imageAddress | STRING | 否       | 图片地址     |               |
| user         | STRING | 否       | 联系人       |               |
| mobile       | STRING | 否       | 联系电话     |               |
| company      | STRING | 否       | 上报单位     |               |
| mUser        | STRING | 否       | 运维人员     |               |
| mMobile      | STRING | 否       | 联系电话     |               |
| shipId       | INT    | 否       | 船的ID       |               |
| watersId     | INT    | 否       | 水域ID       |               |
| userId       | INT    | 是       | 操作人ID     |               |
