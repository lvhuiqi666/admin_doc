# 智慧水域-运维系统-组织架构

## 创建组织建构

### 接口地址：/v1/admin/organ/

### 请求方式：POST

### 请求参数：

| 参数名    | 类型 | 是否必传 | 命名   | 备注 |
| --------- | ---- | -------- | ------ | ---- |
| projectId | INT  | 是       | 项目ID | 必传 |
| userId    | INT  | 是       | 用户ID | 必传 |
| mark      | INT  | 是       | 标记   |      |

### 请求示例:

```bash
{
    "projectId": 4,
    "userId": 3,
    "mark": 1
}
```

### 响应

#### 200

```bash
{
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```
