## 组织架构总览-详情：

> 此接口传入参数， 第一个项目的ID，仅支持单个查询

### 接口地址：/v1/pc/organ/list/

### 请求方式：POST

### 请求参数：

| 参数名     | 类型   | 是否必传 | 命名     | 备注     |
| ---------- | ------ | -------- | -------- | -------- |
| projectId  | INT    | 是       | 项目ID   | 必传     |
| provinceId | INT    | 否       | 省份ID   | 筛选     |
| cityId     | INT    | 否       | 市ID     | 筛选     |
| areaId     | INT    | 否       | 区域ID   | 筛选     |
| watersId   | INT    | 否       | 水域ID   | 筛选     |
| userName   | STRING | 否       | 用户姓名 | 模糊筛选 |

### 请求示例:

```bash
{"projectId": 3}
```
