# 智慧水域-大屏-登录模块

描述:

> 登录模块

## 发送验证码：

### 接口地址：/v1/pc/sms/

### 请求方式：POST

### 请求参数：

| 参数名 | 类型   | 是否必传 | 命名          | 备注 |
| ------ | ------ | -------- | ------------- | ---- |
| mobile | STRING | 是       | 用户名-手机号 |      |
| pd     | STRING | 是       | 密码          |      |

### 请求示例:

```bash
{
    "mobile":"13069634826",
    "pd": "1234567"
}
```

### 响应

#### 200

```bash
{
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```

## 用户登录：

### 接口地址：/v1/pc/login/

### 请求方式：POST

### 请求参数：

| 参数名     | 类型   | 是否必传 | 命名          | 备注 |
| ---------- | ------ | -------- | ------------- | ---- |
| mobile     | STRING | 是       | 用户名-手机号 |      |
| pd         | STRING | 是       | 密码          |      |
| verifyCode | STRING | 是       | 验证码        |      |

### 请求示例:

```bash
{
    "mobile":"13069634826",
    "pd": "123456",
    "verifyCode": "6666"
}
```
