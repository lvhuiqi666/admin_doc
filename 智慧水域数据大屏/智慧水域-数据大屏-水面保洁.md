# 智慧水域-数据大屏-水面保洁

**字段描述**：

cleaning_type:（水面保洁业务）

```bash
(1, u"水上植物"),
(2, u"生活垃圾"),
(3, u"油污收集"),
(4, u"水下植物")
```

## 水面保洁面积总览(默认年总览)：

> 当前接口返回4中类型

### 接口地址：/v1/pc/water_clean/areas_overall/

### 请求方式：POST

### 请求参数：

| 参数名     | 类型 | 是否必传 | 命名     | 备注                     |
| ---------- | ---- | -------- | -------- | ------------------------ |
| year       | INT  | 否       | 筛选年份 | 默认是当年               |
| projectIds | LIST | 是       | 项目ID   | 登录成功以后返回的项目ID |

### 请求示例:

```bash
{
    "projectIds": [3,4],
    "year": 2021
}
```

### 响应

#### 200

```bash
{
    "data": [
        {
            "cleaning_type": 1,  // 保洁类型
            "areas__sum": 19980.0  // 保洁面积
        },
        {
            "cleaning_type": 2,
            "areas__sum": 16650.0
        },
        {
            "cleaning_type": 3,
            "areas__sum": 6660.0
        },
        {
            "cleaning_type": 4,
            "areas__sum": 11988.0
        }
    ],
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```
