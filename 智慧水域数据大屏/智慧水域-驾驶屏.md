# 智慧水域-驾驶屏

## 获取语音播报内容：

### 接口地址：v1/pc/voice_play/

### 请求方式：POST

### 请求参数：

| 参数名 | 类型 | 是否必传 | 命名     | 备注                           |
| ------ | ---- | -------- | -------- | ------------------------------ |
| detect | INT  | 是       | 语音类型 | 1:驱赶2:提醒3:救援4:宣讲5:交互 |

### 请求示例:

```bash
{
    "detect": 1
}
```

### 响应

#### 200

```bash
// 列表页序号 前端自己排序
{
    "data": [
        {
            "detect": 1,  // 播报类型, 见详情
            "name": "吕品品",  // 编辑人
            "c_time": "2021-11-23T07:30:28.531998Z",  // 编辑时间
            "event": "非法捕鱼",  // 触发事件
            "content": "前方人员请注意，请停止非法捕鱼，湖里的鱼这么可爱，你忍心吃吗?", // 播报内容
            "id": 1
        }
    ],
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```

## 发送语音播报内容：

### 接口地址：v1/pc/send_voice_play/

### 请求方式：POST

### 请求参数：

| 参数名  | 类型   | 是否必传 | 命名     | 备注 |
| ------- | ------ | -------- | -------- | ---- |
| msg     | STRING | 是       | 播放内容 |      |
| shipNum | STRING | 是       | 船的编号 |      |

### 请求示例:

```bash
{
    "msg": "前方人员请注意，请停止非法捕鱼，湖里的鱼这么可爱，你忍心吃吗?",
    "shipNum": "0014"
}
```

### 响应

#### 200

```bash
{
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```

## 巡河任务总览接口(驾驶屏图表)：

### 接口地址：v1/pc/patrol/task/overview/

### 请求方式：POST

### 请求参数：

| 参数名    | 类型   | 是否必传 | 命名     | 备注                   |
| --------- | ------ | -------- | -------- | ---------------------- |
| watersIds | LIST   | 是       | 水域ID   | 登录成功后获取水域的ID |
| sTime     | STRING | 是       | 日期     | 默认当天               |
| shipName  | STRING | 否       | 船的名称 |                        |

### 请求示例:

```bash
{
    "watersIds": [2],
    "sTime": "2021-10-16",
    "shipName":"测试"

}
```

### 响应

#### 200

```bash
{
    "data": [
        {
            "distance": "123", // 距离
            "province__lng": "120.193968", // 省份坐标
            "ship__name": "测试船",  // 船的名称
            "waters_id": 2,  // 水域ID
            "c_time": "2021-10-16T16:00:00Z",  // 巡河日期
            "city_id": 1,  // 城市ID
            "ship__number": "00022", // 船的号码
            "city__name": "杭州市",  // 城市名称
            "province__name": "浙江省",  // 省份名称
            "waters__name": "西湖",  // 水域名称
            "ship_id": 2,  // 船的ID
            "province_id": 1,  // 省份ID
            "province__lat": "30.251057", // 省份坐标
            "id": 5
        }
    ],

    "retMsg": "成功 | Success"
}
```
