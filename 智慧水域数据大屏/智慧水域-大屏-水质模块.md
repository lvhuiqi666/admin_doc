# 智慧水域-大屏-水质模块

## 水质历史查询接口：

### 接口地址：v1/pc/history/water_quality/

### 请求方式：POST

### 请求参数：

| 参数名   | 类型   | 是否必传 | 命名     | 备注 |
| -------- | ------ | -------- | -------- | ---- |
| watersId | INT    | 是       | 水域ID   |      |
| sTime    | STRING | 是       | 开始日期 |      |
| eTime    | STRING | 是       | 结束日期 |      |

### 请求示例:

```bash
{
    "watersId": 22,
    "sTime": "2021-06-01",
    "eTime": "2021-06-17"
}
```
