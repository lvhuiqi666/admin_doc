## 业务总览：

> 当前接口总共业务类型目前那有6种，如果近返回3种，那么前端需要用0来进行补充。

### 接口地址：/v1/pc/business/overall/

### 请求方式：POST

### 请求参数：

| 参数名     | 类型 | 是否必传 | 命名     | 备注                     |
| ---------- | ---- | -------- | -------- | ------------------------ |
| year       | INT  | 否       | 筛选年份 | 默认是当年               |
| month      | INT  | 否       | 筛选月份 | 默认是当月               |
| projectIds | LIST | 是       | 项目ID   | 登录成功以后返回的项目ID |

### 请求示例:

```bash
{
    "projectIds": [3,4],
    "month": 9,
    "year": 2020
}
```

### 响应

#### 200

```bash
{
    "data": [
        {
            "business_type": 1,  // 业务类型， 看文档
            "business_type__count": 2  // 数量
        },
        {
            "business_type": 2,
            "business_type__count": 2
        },
        {
            "business_type": 3,
            "business_type__count": 4
        },
        {
            "business_type": 4,
            "business_type__count": 4
        },
        {
            "business_type": 5,
            "business_type__count": 4
        },
        {
            "business_type": 6,
            "business_type__count": 2
        }
    ],
    "retCode": 0,
    "retMsg": "成功 | Success"
}
```
