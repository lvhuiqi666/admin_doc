# APP-巡河记录相关模块

## 获取水下探测任务信息：

> 水域的ID, 根据登录时的账号进行获取

### 接口地址：/v1/wisdom/app/detect/task/list/

### 请求方式：POST

### 请求参数：

| 参数名   | 类型 | 是否必传 | 命名       | 备注 |
| -------- | ---- | -------- | ---------- | ---- |
| watersId | LIST | 是       | 水域ID列表 |      |

### 请求示例:

```bash
{"watersId": [35,58,59]}
```

### 响应

#### 200
